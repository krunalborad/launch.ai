# 🚀 LaunchAI

* **LaunchAI** is a modern **AI-powered SaaS platform** designed to help founders validate startup ideas, build AI products, and launch businesses faster using intelligent automation tools.

# 🌟 Features

## 👤 User Features

* Secure **User Authentication** using Supabase
* **AI-powered startup assistance** tools
* Modern **SaaS-style landing page**
* Interactive **dashboard interface**
* **Protected routes** for authenticated users
* Reusable and modular **UI components**
* Fully **responsive premium UI design**

# 🧰 Tech Stack

### Frontend

* React (Vite)
* TypeScript
* Tailwind CSS
* shadcn/ui
* React Router DOM

### Backend / BaaS

* Supabase (Authentication + Database)
* PostgreSQL (via Supabase)

### Dev Tools

* ESLint
* PostCSS
* TypeScript configuration

# ▶️ Run Locally

### 1️⃣ Install dependencies

```
npm install
```

### 2️⃣ Start development server

```
npm run dev
```

The application runs at:

```
http://localhost:8080
```

LaunchAI is designed for individuals and teams who want to build and launch AI-based products quickly and efficiently.

* **Entrepreneurs** – looking for tools to plan, build, and scale digital products.
